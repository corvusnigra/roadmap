## Зачем

Какие объекты допускаются в модель. Нет «стейкхолдеров» — невидимы конфликты интересов. Нет «жизненного цикла» — поздно вспоминается утилизация. Онтология — это инвентарь типов, между которыми может быть отношение; онтологика — навык не путать тип с экземпляром, факт с интерпретацией, позицию с интересом.

## Ключевые различения

- Тип ↔ экземпляр
- Понятие ↔ референт
- Описание ↔ описываемое
- Факт ↔ интерпретация
- Позиция ↔ интерес
- Роль ↔ исполнитель роли
- Функция ↔ конструкция

- Шаблон онтологического разбора: типы / экземпляры / роли / функции / факты vs интерпретации

- В каждом тексте — отмечать смешение типа и экземпляра
- Раз в неделю — онтологический разбор конфликтной ситуации
- Учебная разметка объекта в день: функция / конструкция / роль / надсистема

- Разводишь типы и экземпляры в разговоре
