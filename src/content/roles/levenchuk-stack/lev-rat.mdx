## Зачем

Не «холодная логика без эмоций». Вероятностное мышление: явные оценки, обновление по Байесу, асимметрия рисков. Калибровка — измеримое качество: когда ты говоришь «80% уверен» — в 80% случаев ты прав.

## Ключевые различения

- Prior ↔ posterior
- Калибровка
- Базовая частота
- Стоимость ошибок (Type I vs Type II)
- Искажения: подтверждение, доступность, нулевой риск, якорь

## Технология

- Журнал прогнозов
- Шаблон: prior / данные / posterior / стоимость ошибки

## Практика (упражнения)

- 3 прогноза в неделю с явной вероятностью; проверка раз в квартал
- «Какова базовая частота?» при сильных утверждениях
- Таблица Type I / Type II в важных решениях

## Маркеры освоения

- 50+ записей в журнале с проверкой
- Замечаешь оценку по доступности в моменте
- Формулируешь асимметрию стоимости в важном решении

## Антипаттерны

- Расплывчатые формулировки прогнозов чтобы не ошибиться
- Перенос байесовского языка в личные разговоры без надобности

## Где в стеке

**Уровень 3: Эмпирика и калибровка.** Иметь дело с реальностью, а не только с моделями. Стержневая часть стека.
